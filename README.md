# Навигатор кампуса УрФУ

Мобильное приложение для навигации по кампусу Уральского федерального университета.

## Функции

- Интерактивная карта кампуса с отмеченными аудиториями
- Поиск аудиторий по названию
- Построение маршрутов между двумя точками
- Переключение между этажами
- Масштабирование и перемещение карты

## Технологии

Apache Cordova, HTML, CSS, JavaScript

## Установка

Скачайте файл app-debug.apk и установите на телефон с Android.

# Краткая инструкция

## Навигатор ИРИТ-РТФ УрФУ

### Задача
Построение маршрутов по новому кампусу ИРИТ-РТФ между аудиториями и другими помещениями, включая переходы между этажами.

### Как пользоваться

1. Выбор точек – нажмите на два маркера на карте: первый – откуда, второй  – куда.
2. Построение – нажмите кнопку «Построить маршрут».
3. Результат – маршрут отображается оранжевой линией. При переходе между этажами у лестницы появляется сообщение «Поднимитесь/Спуститесь на X этаж».
4. Навигация – перетаскивание карты, масштабирование жестами/колёсиком, переключение этажей кнопками 1-2-3.
5. Поиск – введите название аудитории или «столовая» в строку поиска.
