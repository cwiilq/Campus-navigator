# Навигатор кампуса УрФУ

Мобильное приложение для навигации по кампусу Уральского федерального университета.

## Функции

- Карта кампуса с отмеченными корпусами
- Поиск зданий и аудиторий
- Построение маршрутов
- Добавление новых точек на карту

## Технологии

Apache Cordova, HTML, CSS, JavaScript

## Установка

Скачайте файл `app-debug.apk` и установите на телефон с Android.
