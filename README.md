# Навигатор кампуса УрФУ

Мобильное приложение для навигации по кампусу Уральского федерального университета.

## Функции

- Интерактивная карта кампуса с отмеченными корпусами и аудиториями
- Поиск зданий и аудиторий по названию
- Построение маршрутов между любыми двумя точками на карте
- Отображение маршрута поверх плана этажа
- Добавление новых точек на карту через интерфейс приложения
- Переключение между этажами

## Технологии

- Apache Cordova
- HTML5
- CSS3
- JavaScript (ES6)
- SVG для визуализации маршрутов
- Gradle
- Android SDK

## Установка

Скачайте файл `app-debug.apk` и установите на телефон с Android. Для установки может потребоваться разрешить установку из неизвестных источников.
